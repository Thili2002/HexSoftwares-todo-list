# To-Do List Web App

A simple and interactive **To-Do List** web application built with **HTML, CSS, and JavaScript**. Users can add tasks, mark them as completed, and filter tasks by their status.

## Features

- Add new tasks
- Mark tasks as completed
- Filter tasks by **All**, **Pending**, or **Completed**
- Responsive and user-friendly design
- Modern UI with **Glassmorphism** effect

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
